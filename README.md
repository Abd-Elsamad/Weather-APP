# 🌦️ Weather App

A modern and responsive weather application built with **HTML, CSS, and JavaScript** that provides real-time weather information using a weather API. Users can search for any city worldwide and instantly view current weather conditions through a clean, intuitive, and responsive interface.

---

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature
* ☁️ Weather condition and description
* 💧 Humidity information
* 💨 Wind speed
* 📍 Location-based weather
* 📱 Fully Responsive Design
* ⚡ Fast and user-friendly interface
* 🎨 Modern UI with smooth interactions
* ❌ Error handling for invalid locations

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API
